# Biome terrain cuts

Cut from the "Ninja Adventure" asset pack by Pixel-boy & AAA (pixel-boy.itch.io) — CC0.
Same pack the Ruins props came from (`content/map/tileset_floor.png`).

- sand-field / sand-pan / sand-ring / sand-mound: desert floor + features, club bunkers.
- mud-field / mud-blob / mud-strip: bog shore, pool basins, wallows.

Patches were flood-keyed (background removed) so they overlay any ground.
